# 24BCA570P: Client Side Scripting - Practical Syllabus

**BCA Semester-IV**  
**Credits:** L-0 T-0 P-4 C-2  
**Total Hours:** 60

---

## Course Objectives

1. To know about basic understanding of JavaScript's
2. To learn about control flow structures in JavaScript
3. To learn JavaScript functions, objects, and arrays to solve problems
4. To know about higher-order functions in JavaScript to achieve code abstraction
5. To learn the fundamentals of objects in JavaScript using encapsulation, prototypes, classes, and inheritance

---

## Course Outcomes

| Code | Course Outcome |
|------|----------------|
| CO1 | Understand JavaScript's basic elements, including values, data types, and operators |
| CO2 | Apply JavaScript's program structure and control flow concepts to write code effectively |
| CO3 | Analyze and apply higher-order functions for data transformation and abstraction in JavaScript |
| CO4 | Evaluate JavaScript functions, data structures, and recursion to solve practical programming problems |
| CO5 | Create JavaScript programs and classes, implementing object-oriented principles, inheritance, and polymorphism |

## Unit 1: Basic JavaScript Operations (12 Hours)

**Focus:** Introduction to JavaScript syntax, Performing arithmetic operations, Using built-in Math functions, variable manipulation

### Relevant Experiments
1. JavaScript Program to Print Hello World
2. JavaScript Program to Add Two Numbers
3. JavaScript Program to Find the Square Root
4. JavaScript Program to Calculate the Area of a Triangle
5. JavaScript Program to Swap Two Variables
6. JavaScript Program to Convert Celsius to Fahrenheit
7. JavaScript Program to Generate a Random Number

---

## Unit 2: Conditional Statements and Loops (12 Hours)

**Focus:** Using Math.random(), if-else statements to check conditions, modulus operator

### Relevant Experiments
8. JavaScript Program to Check if a Number is Positive, Negative, or Zero
9. JavaScript Program to Check if a Number is Odd or Even
10. JavaScript Program to Find the Largest Among Three Numbers
11. JavaScript Program to Check Leap Year
12. JavaScript Program to Generate a Random Number Between Two Numbers

---

## Unit 3: String Manipulation and Functions (12 Hours)

**Focus:** String manipulation and functions in JavaScript

### Relevant Experiments
13. JavaScript Program to Check Whether a String is Palindrome or Not
14. JavaScript Program to Replace Characters of a String
15. JavaScript Program to Reverse a String
16. JavaScript Program to Convert the First Letter of a String into Uppercase

---

## Unit 4: Array and Date Operations (12 Hours)

**Focus:** Working with arrays and date operations in JavaScript

### Relevant Experiments
17. JavaScript Program to Display Date and Time
18. JavaScript Program to Display Current Date
19. JavaScript Program to Create Countdown Timer
20. JavaScript Program to Remove Specific Item from an Array
21. JavaScript Program to Merge Two Arrays and Remove Duplicate Items
22. JavaScript Program to Sort Array of Objects by Property Values

---

## Unit 5: Advanced JavaScript Concepts (12 Hours)

**Focus:** Advanced JavaScript concepts and operations

### Relevant Experiments
23. JavaScript Program to Get File Extension
24. JavaScript Program to Check If a Variable Is undefined or null
